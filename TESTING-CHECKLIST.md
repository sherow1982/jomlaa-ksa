# ✅ قائمة الاختبار - متجر السعودية

## 🤖 اختبار الشات بوت

### الوظائف الأساسية
- [ ] يظهر زر الشات بوت في أسفل يمين الشاشة
- [ ] النقر على الزر يفتح نافذة الشات
- [ ] النقر على X يغلق نافذة الشات
- [ ] تظهر رسالة الترحيب عند الفتح

### البحث عن المنتجات
- [ ] كتابة "ابحث عن خيمة" يعرض نتائج
- [ ] كتابة "اريد داش كام" يعرض المنتج
- [ ] كتابة "عندكم سكوتر" يعرض النتائج
- [ ] عرض حتى 3 منتجات كحد أقصى
- [ ] عرض الاسم والسعر والتقييم

### معلومات الشحن
- [ ] كتابة "شحن" تعرض معلومات الشحن
- [ ] كتابة "توصيل" تعرض معلومات التوصيل
- [ ] المعلومات صحيحة وكاملة

### سياسة الإرجاع
- [ ] كتابة "ارجاع" تعرض سياسة الإرجاع
- [ ] كتابة "استرجاع" تعرض السياسة
- [ ] المعلومات واضحة

### الأسعار والضرائب
- [ ] كتابة "ضريبة" تعرض معلومات الضرائب
- [ ] كتابة "سعر" تعرض معلومات الأسعار

### التواصل
- [ ] كتابة "تواصل" تعرض معلومات الاتصال
- [ ] كتابة "واتساب" تعرض الرقم
- [ ] الأرقام والبريد صحيحة

### الإحصائيات
- [ ] كتابة "كم عدد المنتجات" تعرض الإحصائيات
- [ ] الأرقام صحيحة (156 منتج)

### الرد الافتراضي
- [ ] كتابة أي شيء آخر تعرض قائمة المساعدة
- [ ] القائمة تحتوي على جميع الخيارات

## 🔍 اختبار مربع البحث

### الوظائف الأساسية
- [ ] مربع البحث يظهر في الهيدر
- [ ] يمكن الكتابة في مربع البحث
- [ ] أيقونة البحث 🔍 تظهر

### البحث الفوري
- [ ] كتابة حرفين يبدأ البحث
- [ ] النتائج تظهر فوراً
- [ ] عرض حتى 5 نتائج
- [ ] كل نتيجة تحتوي على:
  - [ ] صورة المنتج
  - [ ] اسم المنتج
  - [ ] السعر
  - [ ] التقييم

### التفاعل
- [ ] النقر على نتيجة ينقل للمنتج
- [ ] النقر خارج النتائج يغلقها
- [ ] مسح النص يخفي النتائج

## 📱 اختبار قائمة الهمبرجر

### الوظائف الأساسية
- [ ] أيقونة الهمبرجر تظهر على الموبايل
- [ ] النقر على الأيقونة يفتح القائمة
- [ ] القائمة تنزلق من اليمين
- [ ] خلفية شفافة تظهر

### المحتوى
- [ ] جميع الروابط موجودة:
  - [ ] الرئيسية
  - [ ] من نحن
  - [ ] اتصل بنا
  - [ ] سياسة الشحن
  - [ ] سياسة الاسترجاع
  - [ ] سياسة الخصوصية
  - [ ] السلة (مع العداد)

### التفاعل
- [ ] النقر على رابط يغلق القائمة وينقل للصفحة
- [ ] النقر على X يغلق القائمة
- [ ] النقر على الخلفية يغلق القائمة
- [ ] عداد السلة يعمل بشكل صحيح

## 📱 اختبار التصميم المتجاوب

### الهواتف الصغيرة (320px)
- [ ] جميع العناصر تظهر بشكل صحيح
- [ ] النصوص قابلة للقراءة
- [ ] الأزرار قابلة للنقر
- [ ] لا يوجد تمرير أفقي

### الهواتف المتوسطة (375px)
- [ ] التصميم متناسق
- [ ] المسافات مناسبة
- [ ] الصور واضحة

### الهواتف الكبيرة (425px)
- [ ] شبكة المنتجات (2 أعمدة)
- [ ] التصميم متوازن

### التابلت (768px)
- [ ] شبكة المنتجات (3 أعمدة)
- [ ] قائمة الهمبرجر تختفي
- [ ] القائمة العادية تظهر
- [ ] مربع البحث أكبر

### اللابتوب (1024px)
- [ ] شبكة المنتجات (4 أعمدة)
- [ ] التصميم احترافي
- [ ] جميع العناصر متناسقة

### الشاشات الكبيرة (1440px+)
- [ ] المحتوى في المنتصف
- [ ] لا يوجد تمدد زائد
- [ ] التصميم متوازن
