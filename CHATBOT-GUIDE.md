# 🤖 دليل استخدام الشات بوت الذكي

## نظرة عامة

الشات بوت الذكي هو مساعد افتراضي متطور يساعد العملاء في:
- البحث عن المنتجات
- الحصول على معلومات الشحن
- معرفة سياسة الإرجاع
- الاستفسار عن الأسعار
- التواصل مع المتجر

## 🎯 الأسئلة المدعومة

### 1. البحث عن المنتجات
**أمثلة:**
- "ابحث عن خيمة"
- "اريد منتجات للسيارة"
- "عندكم داش كام؟"
- "موجود سكوتر؟"

**النتيجة:**
- عرض حتى 3 منتجات مطابقة
- اسم المنتج
- السعر
- التقييم

### 2. معلومات الشحن
**أمثلة:**
- "كم يستغرق الشحن؟"
- "هل الشحن مجاني؟"
- "معلومات التوصيل"
- "متى يصل الطلب؟"

**النتيجة:**
```
🚚 معلومات الشحن:
✅ شحن مجاني لجميع المنتجات
✅ التوصيل خلال 1-3 أيام عمل
✅ الشحن من داخل سلطنة عُمان
```

### 3. سياسة الإرجاع
**أمثلة:**
- "كيف ارجع المنتج؟"
- "سياسة الاسترجاع"
- "هل يمكن الإرجاع؟"
- "شروط الإرجاع"

**النتيجة:**
```
🔄 سياسة الإرجاع:
✅ إرجاع مجاني خلال 14 يوم
⚠️ لا نقبل إرجاع منتجات العناية الشخصية
```

### 4. الأسعار والضرائب
**أمثلة:**
- "هل توجد ضرائب؟"
- "كم الأسعار؟"
- "هل السعر شامل الضريبة؟"

**النتيجة:**
```
💰 معلومات الأسعار:
✅ لا توجد ضرائب
✅ الأسعار شاملة كل شيء
✅ عروض خاصة على منتجات مختارة
```

### 5. التواصل
**أمثلة:**
- "كيف اتواصل معكم؟"
- "رقم الواتساب"
- "البريد الإلكتروني"
- "معلومات الاتصال"

**النتيجة:**
```
📞 التواصل معنا:
📱 واتساب: +201110760081
📧 البريد: sherow1982@gmail.com
🌐 الموقع: https://jomla-ksa-store.com
```

### 6. إحصائيات المتجر
**أمثلة:**
- "كم عدد المنتجات؟"
- "كم منتج عندكم؟"
- "معلومات المتجر"

**النتيجة:**
```
📊 إحصائيات المتجر:
📦 156 منتج متوفر
🏷️ 35 فئة مختلفة
⭐ تقييمات السعوديةية حقيقية
```

## 💡 نصائح للاستخدام

### للحصول على أفضل النتائج:

1. **كن محدداً في سؤالك**
   - ❌ "منتجات"
   - ✅ "ابحث عن خيمة تخييم"

2. **استخدم كلمات مفتاحية**
   - "ابحث"، "اريد"، "عندكم"، "موجود"

3. **اسأل سؤالاً واحداً في كل مرة**
   - ❌ "كم السعر وكيف الشحن والإرجاع؟"
   - ✅ "كم يستغرق الشحن؟"

4. **استخدم اللغة العربية**
   - الشات بوت مُحسّن للغة العربية

## 🐛 حل المشاكل

### المشكلة: الشات بوت لا يظهر
**الحل:**
1. تأكد من استيراد المكون في Layout.js
2. تحقق من وجود ملف Chatbot.js
3. افحص console للأخطاء

### المشكلة: البحث لا يعمل
**الحل:**
1. تأكد من وجود ملف jomla-ksa-products.json
2. تحقق من صحة البيانات
3. افحص دالة findProducts

### المشكلة: التصميم غير صحيح
**الحل:**
1. تأكد من تحميل CSS
2. افحص media queries
3. امسح الكاش
