# ⚡ البدء السريع - متجر السعودية v2.0

## 🚀 في 3 خطوات فقط!

### 1️⃣ التثبيت
```bash
npm install
```

### 2️⃣ التشغيل
```bash
npm run dev
```

### 3️⃣ الاستمتاع
افتح: `http://localhost:3000` 🎉

## 🎯 جرب الميزات الجديدة

### 🤖 الشات بوت
1. انقر على 💬 في أسفل اليمين
2. اكتب: "ابحث عن خيمة"
3. شاهد النتائج الفورية!

**أسئلة مقترحة:**
```
- ابحث عن داش كام
- كم يستغرق الشحن؟
- كيف ارجع المنتج؟
- هل توجد ضرائب؟
- كيف اتواصل معكم؟
```

---

### 🔍 مربع البحث
1. اكتب في مربع البحث بالهيدر
2. شاهد النتائج الفورية
3. انقر على أي منتج

**جرب البحث عن:**
```
- خيمة
- سيارة
- سكوتر
- كريم
- زيت
```

---

### 📱 قائمة الهمبرجر
1. افتح الموقع على الموبايل (أو صغّر النافذة)
2. انقر على ☰ في أعلى اليسار
3. تصفح القائمة

**الروابط المتاحة:**
- 🏠 الرئيسية
- ℹ️ من نحن
- 📞 اتصل بنا
- 🚚 سياسة الشحن
- 🔄 سياسة الاسترجاع
- 🔒 سياسة الخصوصية
- 🛒 السلة

## 📱 اختبار التصميم المتجاوب

### على الكمبيوتر
1. افتح أدوات المطور (F12)
2. اختر Device Toolbar
3. جرب الأحجام المختلفة:
   - iPhone SE (375px)
   - iPhone 12 Pro (390px)
   - iPad (768px)
   - Desktop (1024px+)

## 🎨 التخصيص السريع

### تغيير ألوان الشات بوت
```css
/* في components/Chatbot.js */
.chatbot-button {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  /* غيّر الألوان هنا */
}
```

### تغيير عدد نتائج البحث
```javascript
// في components/SearchBox.js
.slice(0, 5) // غيّر 5 للعدد المطلوب
```

### تغيير الرسالة الترحيبية
```javascript
// في components/Chatbot.js
const [messages, setMessages] = useState([
  { type: 'bot', text: 'رسالتك هنا' }
]);
```

## 🐛 حل المشاكل السريع

### الشات بوت لا يظهر؟
```bash
# تحقق من:
1. استيراد Chatbot في Layout.js
2. وجود ملف Chatbot.js
3. console للأخطاء (F12)
```

### البحث لا يعمل؟
```bash
# تحقق من:
1. وجود ملف jomla-ksa-products.json
2. صحة البيانات
3. console للأخطاء
```

### التصميم غير صحيح؟
```bash
# جرب:
1. امسح الكاش (Ctrl+Shift+R)
2. أعد تشغيل المشروع
3. تحقق من globals.css
```
